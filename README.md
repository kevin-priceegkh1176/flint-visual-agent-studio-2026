## Usage

Open the studio in a browser, place nodes on the canvas, and connect them to define how the agent should run. From there, tune each step so the workflow matches the behavior you want.

A typical workflow looks like this:

1. Add the nodes you need.
2. Link the nodes into a usable flow.
3. Review the structure in the editor.
4. Update the workflow as your agent changes.

Because the interface is visual, it is useful for exploring ideas, refining logic, and keeping the agent structure easy to inspect.
